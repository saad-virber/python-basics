Python Basics: A Beginner's Guide

Introduction

Python is a high-level, interpreted, and general-purpose programming language. It’s known for its simple and readable syntax, making it an ideal first language for beginners. Python is widely used in web development, automation, data analysis, machine learning, and more.

Python code is typically run using the python or python3 command in a terminal or through an IDE like PyCharm, VSCode, or Jupyter Notebook.

Getting Started

Hello, World

To print text to the screen:

python

print("Hello, World")

Variables

Variables are used to store values. In Python, you don’t declare types—they are inferred automatically.

python

name = "Alice" age = 30 height = 1.75 is_student = True

Variable naming rules:

  • Start with a letter or underscore (_)
  • Can contain letters, numbers, and underscores
  • Case-sensitive (myVar is different from myvar)

Constants

Python has no built-in constant type. By convention, constants are written in ALL_CAPS:

python

PI = 3.14159

This value can still be changed, but you shouldn’t unless necessary.

Data Types

Common built-in types:

  • Integer (int): 5
  • Float (float): 3.14
  • String (str): "hello"
  • Boolean (bool): True, False
  • NoneType: None (represents “nothing”)

Collection types:

  • List: my_list = [1, 2, 3]
  • Tuple: my_tuple = (1, 2)
  • Set: my_set = {1, 2, 3}
  • Dictionary: my_dict = {"key": "value"}

Conditional Statements

Use if, elif, and else to control flow:

python

x = 10 if x > 0: print("Positive") elif x == 0: print("Zero") else: print("Negative")

Indentation defines code blocks—use 4 spaces (not tabs).

Loops

while Loop

Executes as long as the condition is True:

python

n = 0 while n < 3: print(n) n += 1

for Loop

Used to iterate over a sequence (like a list):

python

for i in range(3): print(i)

range(3) generates numbers 0, 1, 2.

Functions

Defined usingdef:

python

def greet(name): print("Hello, " + name) greet("Alice")

You can return values:

python

def square(n): return n * n

Function arguments are passed by value for primitives (numbers, strings, etc.) and by reference for mutable objects (like lists or dicts).

Variable Scope

  • Local variables exist only inside functions
  • Global variables are defined outside all functions and accessible globally.

    • python

    x = 5 # global def example(): x = 10 # local print(x) example() print(x) # still 5

    To modify a global variable inside a function:

    python

    def change(): global x x = 42

Constants With Objects

Even if a constant refers to an object, you can still change the object’s contents:

python

MY_LIST = [1, 2, 3] MY_LIST.append(4) # allowed

Common Keywords

  • def: Define a function
  • if, elif, else: Conditional blocks
  • while, for, break, continue: Looping and control
  • True, False, None: Special constants
  • import: Load a module
  • return: Exit a function and return a value
  • global: Access a global variable inside a function
